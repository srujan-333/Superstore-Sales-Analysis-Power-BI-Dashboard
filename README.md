# Superstore-Sales-Analysis-Power-BI-Dashboard
This project presents an interactive Power BI dashboard built using the Superstore sales dataset to analyze sales, profit, customer behavior, product performance, regional trends, discounts, and shipping efficiency. The dashboard uses well-defined KPIs and dynamic visualizations to provide actionable business insights, enabling data-driven decision-making through filters, trend analysis, and performance comparisons.

# Key Features
KPI tracking: Total Sales, Total Profit, Profit Margin, Orders, Customers
Sales & profit trend analysis over time
Regional and category-wise performance insights
Customer segmentation analysis
Discount impact on profitability
Shipping mode and delivery performance insights

# Tools & Technologies
Power BI Desktop
DAX Measures
CSV Dataset (Superstore Sales Data)

# Use Case
Ideal for demonstrating business intelligence, data visualization, and analytical skills using Power BI.
